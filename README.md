# SRC Pulse | Luxury Celebration 2026 Collection

**SRC Pulse** is a premium e-commerce platform developed for the 17th Annual SRC Week at Dunkwa-On-Offin Nursing and Midwifery Training College. It transitions the traditional student order process into a high-end, digital luxury experience.

## ✨ Core Features

- **Editorial Home**: A high-contrast landing page with atmospheric UI effects (grain, shimmer, ink blobs).
- **Customization Suite**: Interactive product builder for selecting tones, silhouettes, textures, and custom embroidery.
- **Dynamic Checkout**: Secure form with programme-specific class selection and real-time order summaries.
- **Secure Payments**: Integrated with **Paystack** for reliable GHS transactions.
- **Management Dashboard**: Secure admin panel for tracking orders and sales analytics.

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML5, CSS3 (Custom Luxury Theme), Bootstrap 5.
- **Backend**: Supabase (Real-time Database & Auth).
- **Payments**: Paystack Inline JS.
- **Icons**: Phosphor Icons.

## 📁 Project Structure

- `index.html`: Atmospheric landing page.
- `product.html`: Interactive store/customizer.
- `checkout.html`: Order details and payment.
- `confirmation.html`: Proof of purchase & receipt.
- `admin.html`: Executive order management.
- `js/`: Application logic (`app.js`, `product.js`).
- `css/`: Styling system (`style.css`, `theme.css`, `product.css`).
